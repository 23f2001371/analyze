Welcome to Your Data Processing Hub

This application demonstrates an automated data processing workflow, converting source data from data.xlsx (via data.csv) into a structured result.json file.

The entire process, including linting, execution, and deployment, is managed through a GitHub Actions CI/CD pipeline, publishing the results to GitHub Pages.

The Process

  • Source data: data.xlsx
  • Intermediate: data.csv (committed)
  • Processing script: execute.py
  • Output: result.json
  • Automation: GitHub Actions CI/CD

Key Features

  • Automated data conversion
  • Code quality checks with Ruff
  • Python 3.11+ & Pandas 2.3+ compatibility
  • Results published via GitHub Pages
  • Single-file responsive HTML app (this page!)
